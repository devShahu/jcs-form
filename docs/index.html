JCS Membership Form API

জাতীয় ছাত্রশক্তি সদস্য ফরম - Complete REST API for membership form management with PDF generation and admin dashboard.

Base URL: /api
Format: JSON
Auth: Session Cookies

PDF Generation

Generate beautiful PDFs with Bengali font support using Puppeteer/Browsershot.

Admin Dashboard

Secure admin panel to manage submissions, settings, and generate reports.

Form Validation

Comprehensive server-side validation with detailed error messages.

Authentication

The API uses session-based authentication for admin endpoints. After successful login, a session cookie is set that must be included in subsequent requests.

Important

All admin endpoints require authentication. Include withCredentials: true in your requests to send cookies.

Default Credentials (Development)

Username: admin

Password: admin123

Error Handling

All API responses follow a consistent format. Errors include a success: false flag and a descriptive message.

HTTP Status Codes

200 Success - Request completed successfully
400 Bad Request - Invalid input or validation error
401 Unauthorized - Authentication required or invalid
404 Not Found - Resource doesn't exist
500 Server Error - Internal server error

Error Response Format

{
  "success": false,
  "message": "Error description here",
  "errors": {
    "field_name": "Specific field error"
  }
}

POST /api/validate Public

Validate Form Data

Validate form data for a specific step or the entire form. Returns validation errors if any.

Request Body

  "step": "personal",
  "data": {
    "name_bangla": "আব্দুল করিম",
    "name_english": "ABDUL KARIM",
    "father_name": "মোহাম্মদ আলী",
    "mother_name": "ফাতেমা বেগম",
    "mobile_number": "01712345678",
    "blood_group": "A+",
    "nid_birth_reg": "1234567890123",
    "birth_date": "01-01-2000"
  }
}

Success Response

{
  "success": true,
  "errors": {},
  "data": { /* sanitized data */ }
}

Error Response

{
  "success": false,
  "errors": {

GET /api/download/{id} Public

Download PDF

Download the generated PDF for a submission by its ID.

Path Parameters

id string The submission ID returned from form submission

Response

Content-Type: application/pdf

Content-Disposition: attachment; filename="jcs_membership_{id}.pdf"

POST /api/admin/login Admin

Admin Login

Authenticate as an admin user. Returns a session token and sets a session cookie.

Request Body

GET /api/admin/search Auth Required

Search Submissions

Search submissions by name, mobile number, or other fields with optional date filtering.

Query Parameters

query string Search term (name, mobile, etc.)
date_from string Start date (YYYY-MM-DD)
date_to string End date (YYYY-MM-DD)
page integer Page number (default: 1)
limit integer Items per page (default: 20)

Example Request

GET /api/admin/search?query=আব্দুল&date_from=2024-01-01&page=1&limit=10
